# Chatbot using Python 

## Description
This project is a Python-based chatbot developed to interact with users and respond to common queries.  
It works like a mini chatbot by understanding user inputs and giving appropriate responses.

## Features
- Interactive conversation with users
- Handles greetings like Hi, Hello
- Provides current date and time
- Performs basic arithmetic operations
- Gives fallback responses for unknown questions

## Technologies Used
- Python
- Datetime module

## How to Run
1. Install Python on your system
2. Download or clone this repository
3. Open terminal or command prompt
4. Run the chatbot using:
   python chatbot.py
5. Start chatting with the bot

## Output
The chatbot responds intelligently to user queries and performs basic operations successfully.
